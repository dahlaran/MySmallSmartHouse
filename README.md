# MySmallSmartHouse
A small smart home application, able to steer devices such as lights, roller shutters, or heaters.


Language:
- Kotlin
- XML

Architecture:
- MVVM
- Clean Architecture

Techno:
- Binding
- Retrofit
- Room database
- Flow (for the Repository)
- LiveData (for viewModels/UI)
- Hilt

Unit Test:
- JUnit
